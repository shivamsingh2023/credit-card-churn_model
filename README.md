# Credit Card Churn Prediction using TensorFlow

## 📌 Project Overview
This project predicts whether a bank customer will exit (churn) using
Artificial Neural Networks built with TensorFlow.

## 📊 Dataset Features
- CreditScore
- Geography
- Gender
- Age
- Tenure
- Balance
- NumOfProducts
- HasCrCard
- IsActiveMember
- EstimatedSalary

Target:
- Exited (0 = No, 1 = Yes)

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn

## 🚀 How to Run

```bash
pip install -r requirements.txt
python churn_model.py
